# Evaluation Framework

## What to measure

- precision@k
- recall@k
- citation coverage
- confidence calibration
- framework coverage

## Benchmark shape

Each benchmark item should include:

- query text
- expected document ids
- optional filters

## Practical evaluation goals

- attachment queries should surface attachment theory sources
- CBT queries should surface cognitive distortions and reframing sources
- Jungian queries should surface shadow and individuation sources
- emotional regulation queries should surface regulation and neuroscience sources

## Failure modes to watch

- over-weighting lexical overlap
- ignoring publication quality
- returning uncited context
- drifting into diagnosis or therapy language
